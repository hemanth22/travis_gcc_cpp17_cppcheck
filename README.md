The goal of this project is to have a clean Travis CI build, with specs:
 * Build system: none
 * C++ compiler: `gcc`
 * C++ version: `C++17`
 * Libraries: `STL` only
 * Code coverage: none
 * Static code analysis: `cppcheck`
 * Source: one single file, `main.cpp`
